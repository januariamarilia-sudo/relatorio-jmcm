# relatorio-jmcm

Aplicativo Streamlit para gerar o relatório diário de produtividade em Word.

## Arquivos

- `streamlit_app.py`: tela do app online.
- `app.py`: motor que gera o `.docx`.
- `modelo_relatorio.docx`: modelo Word usado como base.
- `requirements.txt`: dependências para o Streamlit instalar.

## Publicar no Streamlit Community Cloud

1. Crie um repositório no GitHub chamado `relatorio-jmcm`.
2. Envie estes arquivos para o repositório.
3. Acesse `https://share.streamlit.io/`.
4. Clique em **Create app**.
5. Selecione o repositório `relatorio-jmcm`.
6. Em **Main file path**, use `streamlit_app.py`.
7. Clique em **Deploy**.

Depois disso, o Streamlit cria um link público para acessar de qualquer computador.
